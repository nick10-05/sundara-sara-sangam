# Background Images for Slokas

This folder contains background images for each sloka page.

## How to Add Ramayana Images

1. **Get Ramayana-themed images** from:
   - Your own collection
   - Free image sites (Pixabay, Unsplash, Pexels)
   - Search for: "Ramayana", "Hanuman", "Indian temple", "ancient India", etc.

2. **Name your images** according to the sloka ID:
   - `sloka-11.jpg` for Sloka 11
   - `sloka-12.jpg` for Sloka 12
   - ... and so on

3. **Supported formats**: `.jpg`, `.jpeg`, `.png`, `.webp`

4. **Recommended size**: 1920x1080 pixels or larger for best quality

5. **Place images here**: `public/images/`

## Example

If you have an image for Sloka 11, place it as:
```
public/images/sloka-11.jpg
```

The application will automatically use local images if they exist, otherwise it will use Unsplash images as fallback.
